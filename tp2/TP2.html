2A SRI TAP - TP2
  Classification de voyelles par lois gaussiennes

Objectifs du TP, savoir :

  • vérifier le type des données à traiter,
  • estimer les paramètres de lois gausiennes,
  • réaliser une classification par cette approche Bayésienne,
  • représenter visuellement les résultats.

Vérification des données

  • Charger les données suivantes : load app_aa ; load app_uu ; load app_ii ;
    Il s'agit de matrices 80x2 résultant d'une ACP sur une paramétrisation cepstrale (cf. TP1).

  • Afficher l'histogramme de chacune des classes : aa, uu et ii. Que pouvez-vous conclure ?

Estimation de lois gaussiennes

  • Estimer les paramètres (moyenne et matrice de covariance) de chaque classe.

  • Ecrire une fonction classer qui renvoie la classe obtenue par maximum de vraisemblance sur les lois gaussiennes estimées sur l'observation fournie en entrée :
    function no_classe = classer(observation,moyenne_aa,covariance_aa, ...
                       moyenne_uu,covariance_uu,moyenne_ii,covariance_ii) ;

    Des observations à tester peuvent être chargées de cette façon : load tst_aa ; tst_uu ; tst_ii ;
    Exemple de résultat : no_classe=classer(tst_aa(1,:),m_aa,c_aa,m_uu,c_uu,m_ii,c_ii)
    => no_classe = 1

Représentation des résultats

  • Représenter en 2D les nuages de points par des couleurs différentes.

  • Afficher les centroïdes des différentes classes.

  • Réaliser une fonction qui effectue tous les tests de classification et qui affiche une matrice de confusion et un taux d'erreur.
    [matrice_confusion,taux_reco]=tout_tester(m_aa,c_aa,m_uu,c_uu,m_ii,c_ii)

  • Commenter les résultats obtenus :
    matrice_confusion =
     20  0  0
      0 20  0
      0  0 20
    taux_reco = 1

  • Réaliser une animation pour l'affichage des points de test (un par un avec des couleurs par exemple...).
